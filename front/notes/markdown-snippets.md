# Markdown Snippets Cheat Sheet

Use these ready-made snippets in the DEV.to editor (or anywhere Markdown is supported). Copy/Paste the blocks you need.

---

## Headings
```
# H1 – Main Title
## H2 – Section Title
### H3 – Subsection
```

## Callouts / Blockquotes
```
> **Note:** Quick highlight.
> - You can stack multiple lines.
```

## Bullet & Numbered Lists
```
- First point
- Second point
  - Nested item

1. Step one
2. Step two
   1. Sub-step
```

## Inline Emphasis & Code
```
**bold text**
*italic text*
`inline code`
```

## Code Fence (with language)
````
```ts
function greet(name: string) {
  console.log(`Hello ${name}`);
}
```
````

Copy the outer triple backticks too; they define the fenced block.

## Multi-language Code Template
````
```bash
npm install
npm run start
```

```html
<section>
  <h2>Section</h2>
</section>
```
````

## Task List
```
- [ ] Pending item
- [x] Completed item
```

## Tables
```
| Column A | Column B |
|----------|----------|
| Item 1   | Detail   |
| Item 2   | Detail   |
```

## Horizontal Rule
```
---
```

## Image Embed
```
![Alt text](https://example.com/image.png)
```

## Link
```
[Angular Docs](https://angular.dev)
```

---

### Quick Insert Macro

````

## Key Steps

2. Step two – description.

```ts

```

## Resources
- [Link 1](https://example.com)
- [Link 2](https://example.com)
````
